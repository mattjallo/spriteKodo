Platforms:
  spriteKodo can be compiled on any platform supported by wxWidgets, 
  namely Linux, Windows, or Max OS X.

A make file is included for Linux / GCC based systems.  

make CONFIG=release

This will create a subdirectory GCCRelease that contains the binary.

To install it, log in as root and copy to your bin directory:

su
cp GCCRelease/spritekodo /usr/local/bin
